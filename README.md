# Banker-s-Algorithm-Simulator

Project Overview:

This project is a simple web-based implementation of the Banker’s Algorithm, used in Operating Systems to check whether a system is in a safe state or not.

Features:

- Create Allocation, Max, and Need tables
- Automatic Need calculation
- Safe sequence detection
- Deadlock (unsafe state) detection
- Easy to use interface

Technologies Used:

- HTML
- CSS
- JavaScript
How to Run:-

1. Download or clone the project
2. Open the project folder
3. Open "index.html" in any browser (Chrome recommended)
How to Use:
1. Enter number of processes and resources
2. Click Create Tables
3. Fill Allocation, Max, and Available values
4. Click Calculate Need
5. Click Check Safe State

Output

- Displays Safe Sequence if system is safe
- Shows Unsafe State (Deadlock) if not safe
